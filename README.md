# Dark Souls II Archipelago

A [Dark Souls II](https://en.wikipedia.org/wiki/Dark_Souls_II) mod for the [Archipelago multi-game randomizer](https://archipelago.gg/). Supports both the Vanilla and Scholar of the First Sin versions of the game.

For help, feedback, or contributing consider joining the [Archipelago Discord](https://discord.com/invite/8Z65BR2).

If you just want to **get started** take a look at the [Setup Guide](https://github.com/WildBunnie/Archipelago/blob/dark_souls_2/worlds/dark_souls_2/docs/setup_en.md).
